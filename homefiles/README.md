# Descripiton
- The script takes the .bashrc file and the .bash_aliases file and makes them symbolically links inside the User's home directory.
- These files come with aliases for "clear"ing the in .bashrc file and commadns for "cd"ing in the .bash_aliases file.
### 
- A .bash_ghusername file is included in the repo because I was working on using all three files before I realized I only needed to have one
    - This is also why the script does both the .bashrc and .bash_aliases.
